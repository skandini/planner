# Быстрое исправление проблем

## Проблема 1: "Backend server is not available"

**Решение:** Запустите бэкенд сервер

```powershell
cd backend
.\.venv\Scripts\Activate.ps1
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

Или используйте скрипт:
```powershell
.\start-backend.ps1
```

## Проблема 2: Участники показывают "не доступен"

**Исправлено:** Убрано отображение "Нет доступа" для участников. Теперь:
- ✅ Можно приглашать любых участников без ограничений
- ✅ Роль в календаре показывается только если пользователь является членом
- ✅ Отсутствие роли не означает, что участник недоступен

## Проверка работоспособности

1. **Запустите бэкенд** (см. выше)

2. **Проверьте health endpoint:**
   ```
   http://localhost:8000/api/v1/health
   ```
   Должен вернуть: `{"status":"ok"}`

3. **Обновите страницу фронтенда**

4. **Попробуйте создать событие:**
   - Выберите участников
   - Убедитесь, что нет ошибок в консоли
   - Проверьте, что доступность загружается

## Если проблемы остаются

1. **Проверьте консоль браузера (F12):**
   - Нет ли ошибок CORS
   - Правильно ли формируются URL запросов

2. **Проверьте логи бэкенда:**
   - Нет ли ошибок при обработке запросов
   - Правильно ли работает endpoint доступности

3. **Проверьте настройки CORS:**
   - Файл: `backend/app/core/config.py`
   - Должен включать: `http://localhost:3000`
